# Litigation Case Management App

A minimalist web app for managing litigation cases, events, and notes.

## Run locally

Open `index.html` in a browser.

## Features

- Login with role-based users (Admin/User)
- Dashboard with sortable columns and All/My case toggle
- Case detail page with editable case fields
- Multiple events per case (add/edit/delete)
- Shared notes per case
- Light/Dark theme toggle
